== Description ==

This plugin does not allow people to comment until either 1) the calculated reading time has elapsed, or 2) they have answered a question created by the author correctly, to bypass the wait time.

= Hold Your Horses Comments =

== Installation ==

= Install =

1. Unzip the 'hold-your-horses-comments.zip' file.
2. Upload the 'hold-your-horses-comments' folder (not just the files in it!) to your 'wp-content/plugins' folder. If you're using FTP, use 'binary' mode.
3. If you have trouble installing, see the [Codex](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation) for more helpful info.

= Activate =

To activate, simply visit Dashboard --> Plugins as a site administrator on the backend and hit the "Activate" link.

= Settings = 

Navigate to Settings --> Hold Your Horses on your site's backend and feel free to set the default fields. Note that defaults can be overridden by individual post settings on the edit post screen.
	
= Requirements =

- PHP 5.1 OR ABOVE
- WordPress 3.0 or above

== Documentation ==

The css attributes can be overridden by adding new definitions to your theme style.css file and the defaults can be reviewed from the hyh.css file located in the css folder of the Hold Your Horses Plugin folder.
